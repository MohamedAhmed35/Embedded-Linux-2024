## Create Your First "Hello, World!" Program

### Steps to Create "Hello, World!"

#### 1. Create python file from terminal
>      touch hello.py
#### 2. Write the following code in file:
>      print("Hello, World!")
#### 3. Run the script from the terminal
>      python3 hello.py
